HTML

  • The head element contains information about the webpage.
    • The charset element sets character encoding.
    • The http-equiv element sets an HTTP header for content.
    • The viewport element sets the default view for our webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the webpage's title.
  • The style element defines the CSS styles of the webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border indicates how much space we want around the padding but inside the margin.
  • Declarations are composed of the CSS property we are applying and the value of the property.
  • A selector defines the element or attributes to which the declarations will apply.
  • The rel attribute specifies the relationship between the current document and the linked document resource.

Git

  • git status: checks what branch we are currently on
  • git branch: lists all branches, * will appear next to active branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git config --global user.name "[firstname lastname]": sets name identification for reviewing history
  • git config --global user.email "[valid email]": sets email address assosicated with history marker
  • git pull: merge any commits from remote branch
  • git push [alias]/[branch]: transmit local branch commits to remote branch
  • git merge [alias]/[branch]: merge remote branch into current branch and brings it up to date

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • A string is data within a variable and is always contained in quotation marks.
  • Each line of JavaScript code ends with a semicolon.
  • JavaScript is always read from top to bottom.
  • The strict equality operator checks to see if two values are equal.
  • Conditional statements allow us to interrupt control flow.
  • An array is a single variable that is used to hold a group of data that are typically related.
